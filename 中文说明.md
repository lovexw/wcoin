# 💎 WCOIN - 去中心化挖矿系统

## 一句话介绍

WCOIN是一个用Python实现的类比特币区块链系统，支持P2P网络、工作量证明挖矿、自动减半机制，总量1500万枚，10年产出完毕。

## 🎯 核心特点

| 特性 | 说明 |
|-----|------|
| 💰 总量 | 15,000,000 WCOIN |
| ⏰ 减半周期 | 每年减半一次（52,560个区块） |
| 📅 产出时间 | 10年后停止产出 |
| ⛏️ 初始奖励 | 143 WCOIN/块 |
| 🕐 出块时间 | 约10分钟 |
| 🔐 加密算法 | SHA256 + RSA |

## 🚀 3步快速开始

### 1️⃣ 安装依赖
```bash
pip install -r requirements.txt
```

### 2️⃣ 启动节点
```bash
python main.py
```

### 3️⃣ 查看Dashboard
打开浏览器访问: http://localhost:5000

就这么简单！🎉

## 📱 界面预览

Dashboard实时显示：
- ⛓️ 区块链高度、难度、算力
- 💰 流通量、减半倒计时
- ⛏️ 挖矿状态、奖励统计
- 🌐 网络连接、节点信息
- 📦 最新区块列表

每5秒自动刷新！

## 🌐 多节点网络

### 一键启动3节点网络
```bash
./start_network.sh
```

访问：
- http://localhost:5000 （节点1）
- http://localhost:5001 （节点2）
- http://localhost:5002 （节点3）

### 停止网络
```bash
./stop_network.sh
```

## 🔧 自定义配置

编辑 `config.py` 修改参数：

```python

GENESIS_DIFFICULTY = 2       # 降低难度
HALVING_INTERVAL = 10        # 10块减半

# 生产环境配置
BLOCK_TIME = 600             # 10分钟出块
GENESIS_DIFFICULTY = 5       # 提高难度
HALVING_INTERVAL = 52560     # 1年减半
```

## 🎓 技术栈

- **语言**: Python 3.11+
- **Web框架**: Flask
- **加密库**: PyCryptodome
- **共识机制**: PoW (工作量证明)
- **存储**: JSON文件
- **部署**: Docker支持

## 🔐 安全提示

⚠️ 这是测试版本，请注意：
